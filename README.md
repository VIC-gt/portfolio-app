# Portfolio SPA

A personal project showcase application built with React.

## Features

- View a list of projects on the landing page
- Add new projects using a form
- Search and filter projects by title, tech, or category
- Responsive design

## Tech Stack

- React
- CSS

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
